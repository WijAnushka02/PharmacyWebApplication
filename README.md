# 🏥 Pharmacy Web Application

A comprehensive **Pharmacy Web Application** designed to digitalise and streamline pharmaceutical services. This system provides a secure and user-friendly platform for managing prescriptions, medications, inventory, and administrative operations, supporting patients, pharmacists, and administrators through role-based access.

---

## 🚀 Features

### 👤 Patient Module

* Easy navigation through Home, Medications, Prescriptions, and Support sections
* Upload and manage prescriptions
* View order history and track status
* Search medications by name or category (e.g., diabetes, allergies)
* Maintain personal health records, payment details, and vaccination history

---

### 💊 Pharmacist & Staff Module

* Dashboard with overview of orders, sales, and stock alerts
* Inventory management (add, update, delete medicines)
* Supplier and expiry date tracking
* Prescription validation and order processing

---

### ⚙️ Admin Module

* User management with role-based access (Admin, Pharmacist, Support)
* System configuration and monitoring
* API integration (SMS/Email services)
* Payment gateway configuration
* Audit logs and system activity tracking

---

## 🛠️ Tech Stack

* **Frontend:** HTML, Tailwind CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL (or any relational database)

---

## 🎨 UI/UX Design

* Clean and professional interface using green and white colour scheme
* Fully responsive design for desktop and mobile devices
* Mobile-friendly navigation (hamburger menu, bottom navigation bar)
* Accessibility support with keyboard navigation and ARIA labels

## ⚡ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/WijAnushka02/PharmacyWebApplication.git
```

2. Move the project to your local server directory (e.g., XAMPP `htdocs`)

3. Import the database:

* Open phpMyAdmin
* Create a new database
* Import the provided `.sql` file

4. Configure database connection:

* Update database credentials in the config file

5. Run the application:

* Start Apache and MySQL
* Open browser and go to:

```
http://localhost/PharmacyWebApplication
```
